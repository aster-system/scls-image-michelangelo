# SCLS - Image
## What is SCLS ?
SCLS is an Aster System's set of libraries made to simplify development in C++, with a lot of useful and simple tools made to simplify the life of C++ developers.
## What is SCLS Image ?
SCLS  Image "Michelangelo" is the mathematical part of the SCLS library. It contains a lot of basic (and specific) tools to handle image in C++.
## Requirements
This library is made for C++ 20, but can be used in C++ 17.
This library needs the SCLS Math "Carl" library, which uses the SCLS Foundation "Leonhard" library.
The include path to the SCLS Math headers can be specified in a pre-processor directive : SCLS_MATH_PATH (it must be written before including SCLS Image).
