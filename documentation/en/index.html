What is SCLS ?

SCLS, for "Simple C++ Libraries Set", is a sets of C++ library, which means to make C++ development almost as easy as in Python.

What is SCLS Image ?

SCLS Image allows its users to do some very simple image manipulation in C++. It should also be very light to use, and allows to do complex image manipulation in a few lines.

It is name "Michelangelo" named after one of the greatest painter of all time, Michelangelo.

What to do with SCLS Image ?
